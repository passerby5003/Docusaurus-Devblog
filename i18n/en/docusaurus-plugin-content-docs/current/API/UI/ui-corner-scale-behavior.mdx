# UI Corner Scale Behavior

Basic UI sliding and scaling effect with directionality. The UI moves from the start point to the end point along a default Bezier curve while scaling from (0.1, 0.1) to (1, 1). The UI scales from the four corners (representing the four directions) or according to the set start and end points.

The effect is similar to clicking on an image and having it smoothly enlarge from its current position to fill the entire screen, similar to the effect of opening a preview by pressing the trackpad on a MacBook.

Both the start and end points are based on the anchored position. Ensure careful consideration when specifying these points.

This behavior has two modes: `Synchronize` and `Sequence`.
- `Synchronize`: Perform scaling and sliding simultaneously.
- `Sequence`: Perform sliding first, followed by scaling.

:::tip[More Interesting Effects?]
    Changing the Pivot of the UI coordinates can reveal more interesting effects.
:::

:::warning[Note]
    When the end coordinate is (0, 0), the UI is moved off-screen by default.
:::

## Usage Example
```csharp title="Example.cs"
    ShowUI(new Vector2(1f, 0f), new Vector2(1000f, 0f), UiShowHideType.SlideScale, UiShowHideDirection.Left, 1f, UiShowHideBehavior.Synchronize);
```
