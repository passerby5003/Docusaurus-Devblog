# UI Custom Sequence Behavior

Manually specify UI animations. Unlike the preset classes mentioned earlier, this class does not require inheriting from [UIBehaviorBase](./ui-behavior-base.mdx). Instead, it can directly inherit from [UIFrameworkBase](./ui-framework-base.mdx). You need to write and control the Dotween Sequence yourself to achieve the desired effect.

Inherits from: 
- [IUICustomBehavior](./iui-custom-behavior.mdx)

## Usage Example
```csharp title="Example.cs"
    curSequence = DOTween.Sequence();
    curSequence.Append(search.DOLocalMoveX(0f, 1.5f).Pause());
    curSequence.Join(cam.DOLocalMoveX(100f, 1.75f).Pause());
    curSequence.Append(imgbg.DOColor(new Color32(120,70,100,255), 1.5f).Pause());

    ShowUI(curSequence, this);
```
