# 自我介绍

你好！这里是Rarick，一个 Unity 开发者。目前开发过一些小程序、小游戏，同时也在策划新的游戏，希望能在今后正式地参加一次Game Jam！  
Game Jam 可以说是最想参加的一个活动，既能挑战游戏开发的极限，也是对自己自身能力的一种肯定。不管最后做出来的是什么，都是自己对高强度参与自己喜欢的事物，而开花结果后的成果。（当然第一次参加的话肯定是一脸懵就是了xd）

现在主要活动在江浙沪地区，平时会去一些漫展啊什么的地方玩，有兴趣来找我一起玩叭！

## 教育经历

我毕业于金陵科技学院，一所位于南京的平凡而又默默无名，但又丰富了我的人生的大学。

![img](./collage-logo.png)

学校里的大家都很努力和勤奋，只要自己参与的事情，就一定要做到追求专业和极致，而且在大学城一块区域算是很大的一个学校了，整个学校从南走到北要15分钟（甚至没有班车，只能走）。在计算机系，几个舍友和认识的同学也都互相帮助，分享知识，解决难题。那段一起通宵写报告和反反复复来回审论文的时间虽然劳累，但也是我最难以忘怀的大学生活的点点滴滴。

## 兴趣爱好

震惊！某开发工程师，竟喜欢出去拍照！（雾）

没错，是的你没听错，我还挺喜欢出去拍照的。还有几张照片存档到了[LFI](https://lfi-online.de/en/gallery/Rarick-Wong/Glowing-Light-1959628.html)上（虽然没拿奖就是了，因为懒得P，原图已经很不错了）

平时摄影装备非常简单，就一台相机，最多带个聚光灯。相机镜头参数：
- SummiLux 1:1.7/28mm ASPH.
- F1.7
- 1/36000s max
- 28/35/50mm

另外，我也喜欢吹哨笛（同时也想学长笛，准备以后去找个琴行老师学习一下），虽然气息控制还不是很利索，但是想吹什么小曲基本都能自己吹出来。平时也会拿Fl Studio出来玩玩作为兴趣爱好。乐理目前在学，目前学完了乐理基础（小学到高中的乐理）。

也是因为学习了乐理，我才发现创作的门槛不是在于工具（专业制作人都用的Logic Pro），而是在于扎实的基础和丰富的创作力。人类和AI的最大差别，就是人类拥有AI所不能企及的天马行空般的想象力，正式因为这股想象力，推动了人类的社会和生活水平不断进步。
