# UI Slide Scale behavior

UI 基础的滑动+缩放效果，具有方向性，表现形式为从起点按照默认贝塞尔曲线移动到终点，同时UI会按照0，0到1，1进行缩放。

起点和终点都是基于anchored position，填写起点和终点时务必注意。

该表现形式具有两种模式： `Synchronize` 和 `Sequence`.
- `Synchronize`: 同时执行缩放和滑动
- `Sequence`: 先执行滑动，再进行缩放

:::tip[更有意思的变化？]
    更改UI坐标的Pivot，可以发现更多有趣的效果。
:::

:::warning[注意事项]
    当终点坐标为0，0时，默认移出屏幕外.
:::

## 使用范例
```csharp title="Example.cs"
    ShowUI(new Vector2(1f, 0f), new Vector2(1000f, 0f), UiShowHideType.SlideScale, UiShowHideDirection.Left, 1f, UiShowHideBehavior.Synchronize);
```
