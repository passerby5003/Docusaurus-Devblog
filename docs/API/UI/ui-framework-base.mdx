# UI Framework Base

整个UI框架的基础，UI的所有进入退出、红点系统等，都由此类进行控制。
:::danger[导入时即报错的注意事项]
    务必检查是否正确安装了Dotween（免费或付费版都可以），以及检查是否正确安装了Addresable。  
    如无需红点系统，建议直接将红点相关代码注释掉
:::

## 字段参数
| 名称 | 类型 | 访问级别 | 描述 |
| :-: | :-: | :-: | :-: |
| startShowEvent | Func | private | 开始展现UI时的事件 |
| completeShowEvent | Func | private | 结束展现UI时的事件 |
| startHideEvent | Func | private | 开始隐藏UI时的事件 | 
| completeHideEvent | Func | private | 结束隐藏UI时的事件 |
| startMoveEvent | Func | private | 开始移动时的事件 |
| completeMoveEvent | Func | private | 结束移动时的事件 |
| simpleTween | Tween | private | 一般的移动时所用的Dotween动画 |
| redPointhandle | AsyncOperationHandle | private | 单个UIFramework所指定的专属于此UI的红点 |

:::warning[需要注意的点]
    Func的结构： `Func<object, object>`   
    Func中的第一个参数是Action的默认参数，一般我们用第二个参数，即UI本身（一般是，根据具体环境决定）
:::

:::warning[需要注意的点]
    UI的`ShowUI()`和`HideUI()`方法需要检测UI本身是否激活，因此调用时需要通过父级或者总管脚本进行控制。这样的设计是为了逐层调度UI，避免出现UI 1号直接调了UI 2号。
:::

## 具体使用范例
### UI管理类的范例

```csharp title="UIFrameworkExampleManager.cs"
public class UIManager : MonoBehavior
{
    //参数和字段等...
    public SubUI subUI;

    private void Start() {
        subUI.SetUI(true);
    }

    private void OnApplicationQuit() {
        subUI.SetUI(false);
    }
}
```

### UI框架类的范例  

```csharp title="UIFrameworkExampleSubUI.cs"
public class SubUI : UIFrameworkBase
{
    //参数和字段等...

    //OnEnable和OnDisable等,需要override

    public void SetUI(bool isShow)
    {
        if (isShow)
        {
            //需要移动到的位置要算一下，一般在Hierachy中调到Debug模式看anchored position，这里直接用0，0作为终点
            ShowUI(UiShowHideType.Slide, UiShowHideDirection.Left, 1f, UiShowHideBehavior.Synchronize);
        }
        else
        {
            //需要移动到的位置要算一下，一般在Hierachy中调到Debug模式看anchored position，这里直接用0，0作为终点
            HideUI(UiShowHideType.Slide, UiShowHideDirection.Left, 1f, UiShowHideBehavior.Synchronize);
        }
    }
}
```

### 红点的使用范例
```csharp title="UIFrameWorkExampleRedPoint.cs"
private bool isRead = false;
    protected bool b_IsRead
    {
        get { return isRead; }
        set
        {
            isRead = value;
            //要取反，因为阅读完了后是取消红点，未阅读是激活红点
            UseRedPoint(() => !isRead);
        }
    }
    public override void OnEnable()
    {
        base.OnEnable();
        b_IsRead = false;
        //btn表明的是某一个UI按钮
        btn.onClick.AddListener(() =>
        {
            Camera.main.GetComponent<CameraPerspective>().MoveCameraTo(locator.position, 500f);
            b_IsRead = true;
        });
    }
```
